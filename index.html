Beep Boop Converter

This web app will modify an array of numbers starting with 0 and ending with the number you enter in the 3 following ways:

  1. Replacing any number containing 0 with "Beep!"
  2. Replacing any number containing 1 with "Boop!"
  3. Replacing any number divisible by 3 with "I'm sorry, Dave I'm afraid I can't do that."

Note: These commands are in order of importance, so a number divisble by 3 that also contains a 0 will ignore the 0 command and just perform the divisible by 3 command.
